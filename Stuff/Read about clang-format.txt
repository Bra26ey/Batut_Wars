Чтобы .clang-format работал, необходимо в терминале прописать: sudo apt-get install clang-format
Возможные стили: google, chromium, mozilla, webkit, microsoft, llvm.
Сейчас я записал туда google и отформатировал все файлы.
Использовать автоматическое форматирование: clang-format -i project/src/*.cpp для .cpp и аналогично для .h (указать верный путь). вместо * может стоять отдельный файл.
В vs code если в каталоге есть .clang-format файл, то форматирование кода на пкм->форматировать или [linux]: ctrl + shift + i будет происходить по законам файла .clang-format.
Перед каждым git push предлагаю форматировать, чтобы на гите лежал код в выверенном стиле.
